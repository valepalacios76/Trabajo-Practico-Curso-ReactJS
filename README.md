GlamBeauty
GlamBeauty es una plataforma de e-commerce desarrollada en React como parte de mi curso de React js en Coderhouse.Me llamo Valentina Palacios y esta aplicación está inspirada en la pasión por la moda, la belleza y el cuidado personal, ofreciendo productos de maquillaje, fragancias y muebles de moda

Componentes Principales
NavBar: Barra de navegación principal
ItemListContainer: Contenedor de la lista de productos
ItemList: Componente para mostrar una lista de productos
Item: Componente individual para mostrar cada producto
ItemDetailContainer: Detalles específicos de un producto
ItemDetail: Componente para mostrar los detalles de un producto
ItemCount: Componente para seleccionar la cantidad de productos
Carrito: En ella encontrarás el carrito de compras
Checkout: Página para finalizar la compra
CartWidget: Widget del carrito incluido en la barra de navegación
Footer: Pie de página

Funcionalidades
Navegación: Permite a los usuarios navegar diferentes categorías de productos
Detalles del producto: Visualización detallada de cada producto disponible con su descripción y precio en ella
Gestión del carrito: Funcionalidad para agregar, eliminar productos y finalizar la compra
Finalización de compra sencilla: Proceso para completar y confirmar el pedido

Rutas
/: Página principal con la lista de productos
/item/: Detalles de un producto específico
/category/: Lista de productos de una categoría específica
/carrito/: Detalles de los productos en el carrito
/carrito: Página del carrito de compras
/checkout: Página de finalización de compra

Tecnologías Utilizadas
React: Biblioteca JavaScript para la construcción de interfaces de usuario
React Router: Para la gestión de rutas en la aplicación
Context API: Para la gestión del estado global de la aplicación (CartProvider)
React Hook Form: Biblioteca para la gestión eficiente de formularios
useState: Hook para el manejo del estado local en componentes funcionales
useEffect: Hook para la gestión de efectos secundarios en componentes funcionales
Firebase: Utilizado para la autenticación de usuarios y la base de datos
Netlify: Plataforma para el despliegue y alojamiento de la aplicación
